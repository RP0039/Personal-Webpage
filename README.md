# Personal Website Source

This repository contains the source code of my personal website.

> [!NOTE]
> The live version of the site is hosted on Neocities.
>
> Live site:
> https://rp0039.neocities.org

## About

This project is a personal website where I experiment with web design and development while learning new technologies.

The repository is public so people can explore the code, suggest improvements, or report issues.

## Project structure

src/ <br>
 |-/styles → stylesheets <br>
 |   |- global-css → Components, Media querys and Variables <br>
 |   |- blog.css <br>
 |   |- updates.css <br>
 | <br>
 |-/script → JavaScript <br>
 |-/assets <br>
 |   |- /font → fonts <br>
 |   |- /svg → svgs <br>
 |   |- /img → images <br>
 | <br>
 |-/layout <br>
 |   |- layout.astro → general layout <br>
 |   |- /post → Blog and blog post layouts <br> 
 |-/pages <br>
 |   |- /blog/[slug].astro → dinamic paths <br>
 |   |- blog.astro <br>
 |   |- home.astro <br>
 |   |- extras.astro <br>
 |   |- /extras → extras pages <br>
 |   |- index.astro → initial page <br>
 | <br>
 |-/content → Content of blog, updates and log _**(MARKDOWN only)**_ <br>
 | <br>
 |-/components <br>
