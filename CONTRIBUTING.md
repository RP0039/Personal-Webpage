> # Contributing
> Thank you for taking interest in this project.
>
> This repository contains the source code of a personal website. Contributions are welcome, but please keep the following guidelines in mind.

## Ways to contribute

* Make pull requests clean and simple, maintain the code style, and avoid unnecessary changes.
* Report website issues.
* Left me sugestions in my mail on my profile.
* Sharing my website with your friends.
